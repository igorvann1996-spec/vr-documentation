# 📜 Загальні Положення Командування

Цей розділ є офіційним статутом для Вищого Командування (Спікерів та Офіцерів). Кожен Командувач (РЛ) несе особисту відповідальність перед Урядом Гільдії «Верховна Рада» за підготовку, тактичне планування та успішне проведення військових операцій.
