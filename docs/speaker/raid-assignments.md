# Призначення / Анонси / Особливості

## **Перед початком**

Завжди перед початком рейд лідер мусить задати основні призначення для класів, бафи, та загальні призначення для всіх класів які мають важливий вплив в рейді.

Загальне

* Нагадати за зміну мейн спеку в каналі **\#мс-ченджі**
* Позначити танків, та наголосити якщо вони змінюються
* Назначити інервейти на хілів
* Назначити порядок бафів для паладінів
* Назначити порядок ХОПів для паладінів
* Назначити порядок Мас Сакри для паладінів (прот, хпала). 90% випадків прот паладін дає першим мас сакру
* Назначити порядок Гімнів (хіл, мана) для прістів. Завжди ДЦ / Хпріст повинні бути перші в цьому порядку.
* Поставити рибу
* Роздати фласки
* Пул таймер 65 секунд. Воїни використовують поти на 60 та 0 секундах. У випадках коли воїни не використовують поти, пул таймер скорочується до 25 секунд для щитків ДЦ пріста

## **Lord Marrowgar**

Перед пулом

* Пул таймер 65 секунд, воїни використовують поти на 60 та 0 секундах
* Впевніться, що озвучений порядок для Divine Sacrifice

Під час бою

* Анонс мас сакри коли бос влітає в рейд
* Анонс фаер мастер аури коли бос влідає в рейд
* Анонс збивання шипів
* Анонс рухання за босом

Після закінчення

* Сови та ШП ресають тих хто помер
* Сови роздають назначені інери
