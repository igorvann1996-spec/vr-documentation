# Зміна мейн спеку

Для зміни мейн спеку потрібно **перед початку рейду** записати зміну в каналі **#мс-ченджі** для того щоб РЛ міг зафіксувати це перед рейдом.

Формат запису `{ім’я персонажа} {спек, або один предмет який гравець хоче бідати}`, наприклад:  

```
Logansdk ms frost
Loganshunt ms Bauble of True Blood
```

Гравцю дозволяється бідати будь який предмет якщо він змінив спек саме для цього предмету, та якщо цей предмет **виправданий його потребами**.

**ВАЖЛИВО** Змінювати спек на ПВП не дозволяється.

**ВАЖЛИВО** Якщо ви хочете отримати дублікат предмету, який ви вже маєте, то ви повинні вказати це в каналі **#мс-ченджі** як **МС Чендж на одні річ**.

## Пріоритетність

В гільдії є поняття пріоритетності бідінгу.

Пріоритетність бідінгу визначається наступним чином:

1. Рівний пріоритет в наступних випадках:

   * **Біс ліст** - це список предметів який прописаний в офісійному документі біс лістів - **Біс лісти для всіх класів** [Lordareon BiS Lists](https://docs.google.com/spreadsheets/d/1HZK0H8YdVcQEXj8Lbne3n6nGqKMvmcb4_3ogM4erQrI/edit).
   * **МС Чендж на одні річ** - це зміна мейн спеку, яка була вказана в каналі **#мс-ченджі** виключно під одну річ, наприклад: `Logansdk ms Bauble of True Blood`.


Якщо гравець вказує що зі всього рейду йому потрібен лише один предмет, то він має право бідати лише цей предмет,
і не може бідати інші предмети, що також робить його пріоритетність рівною з іншими гравцями, які можуть бідати цей предмет по **Мейн спеку** або **Біс лісту**.


Наприклад, гравець на ММ хантері вказує в каналі **#мс-ченджі** що йому потрібен лише один предмет - [Signet of Twilight](https://www.wowhead.com/wotlk/item=54576/signet-of-twilight).
Це дозволяє йому бідати цей предмет на рівні з гравцями для який це предмет з **Біс лісту**.

2. **Мейн спек** - це спек, який вибраний класом за замовчуванням, або який, гравець, вказав в каналі **#мс-ченджі**.

Зміна мейн спеку автоматично прирівнюється до **Біс ліст** для гравця, який змінив спек. Для не біс речей **МС Чендж на одні річ** буде вважатись приорітетнішим.

Наприклад, гравець на **прот** воїні змінює спек на **фурі**, при бідінгу [Might of Blight](https://www.wowhead.com/wotlk/item=50693/might-of-blight) в нього буде нижчий пріоритет ніж у гравця на ДК в **фрост** або **анхолі** спеку, тому що це не біс річ для нього.

3. **Офф спек** - це другий спек класа за замовчуванням.

Речі які не є біс речами для будь якого спеку який приймає участь в бідінгу - дозволяються бідати всім на мейн спек.
