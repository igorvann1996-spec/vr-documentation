# Черга на ШМ / Квести

Для того щоб встати в чергу на ШМ, або на квести, потрібно написати будь кому з ГМів ім’я персонажа та кількість шардів, або квестів, та мати звання не менше ніж рейдер.
Формат `{ім’я персонажа} {к-ть шардів}`, наприклад "Logansdk 15".  
При записі в рейд, потірбно обов’язково вказати біля персонажа що він збирає шарди, якщо він записаний в відповідний список, наприклад:  

```
Loganshunt mm 6.5  
Logansdk frost 6.3 - шарди
```

Якщо людина пропустила два або більше РТ, то цей гравець буде виключений зі списку, але може знову встати в список.  
Після кожного рейду, ГМ зообов’язаний змінювати кількість шардів, або квестів в повідомленні в каналі.  
Коли гравець закінчить збирати ШМ, ГМу потрібно оновити список, виключивши цього гравця з нього, та перемістивши інших на позицію вгору.

**Одночасно гравець може збирати шарди лише на один ШМ.**  
**Доступна кількість місць - п’ять.**
