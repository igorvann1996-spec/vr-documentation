# 📜 Посібник для Новобранців

Вітаємо у лавах Збройних Сил гільдії «Верховна Рада»! Цей військовий маніфест створений спеціально для новобранців (рекрутів), щоб допомогти вам швидко адаптуватись до державного ладу та стати повноправними громадянами.

> **Увага:** Перед тим як продовжити, переконайтеся, що ви повністю ознайомились з [Довідником Громадянина (F.A.Q.)](./f-a-q.md).

## 🪪 Ідентифікація у Спецзв'язку
Ваш перший крок — отримання доступу до урядових каналів зв'язку. Для цього вам необхідно змінити своє ім'я на сервері Discord за офіційним стандартом: **Ім'я вашого персонажа (Ваше реальне ім'я)**.
Наприклад:
```
Logans (Олег)
Tipotank (Костя)
```

## 🎖️ Кар'єрний Шлях та Рейди
Військові кампанії нашої держави поділяються на різні рівні складності:

* **Тренувальні табори (до 5.2к GS):** Якщо ваш персонаж ще не має базового екіпірування, вас не мобілізують до елітних військ (ЦЛК / РС / ТОГК). Проте для вас відкриті двері у ТОК 25 / ТОК 10 / ЦЛК 10 (нормал), за умови наявності хоча б **5к GS**, а також базових інчантів та гемів.
* **Резерв в елітних військах:** Якщо командування ухвалило рішення взяти вас в основний рейд, але ви ще не досягли 5.2к GS або не повністю зачаровані/загемовані, ви йдете у статусі **Резервіста**. Це означає, що ви працюєте виключно за заробітну плату (DKP) та баджі. Ви можете претендувати на військові трофеї (loot) лише в тому випадку, якщо вони більше нікому не потрібні.

Якщо у вас лише один боєздатний персонаж або ви фізично не маєте змоги з'являтися у певні дні мобілізації, **обов'язково повідомте про це** під час подачі заявки на РТ. Це допоможе Командувачам (РЛ) ефективніше планувати військові операції.

## 🛠️ Тактичне Забезпечення (Аддони)
Для успішного виконання наказів командування кожен боєць зобов'язаний встановити наступне тактичне програмне забезпечення:

1. **Pally Power** — спеціалізований аддон для Паладінів та Командувачів, необхідний для координації державних бафів (аур).
2. **Deadly Boss Mods (DBM)** — життєво необхідний радар для всіх солдатів, який анонсує ворожі атаки та таймери.
3. **Рейдові Фрейми (Grid2, VuhDo, XPerl тощо)** — інтерфейс для відображення стану здоров'я всього підрозділу, що є критично важливим для ситуаційної обізнаності на полі бою.
