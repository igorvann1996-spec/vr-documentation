# Для новачків

Цей розділ розрахований на новачків, або людей які тільки вступити в гільдію.

**Перед тим як почати читати цей розділ, впевніться що ви повністю ознайомились з [розділом F.A.Q.](./f-a-q.md).**

Для самого початку потрібно отримати права в діскорді. Якщо ви читаєте цю документацію, отже ви це мали вже зробити, як ні, то потрібно змінити ім'я на сервері на ім'я свого персонажу (мейна) і в дужках своє ім'я. Наприклад:

```
Logans (Олег)
Tipotank (Костя)
``` 

Якщо ваш основний персонаж менше 5.2к гс, це означає що наразі важко буде вас приймати, або брати в основні рейди, такі як ЦЛК / РС / ТОГК, але це не означає що ви нікуди не попадете.
В такі рейди як ТОК25 / ТОК10 / ЦЛК10 нормал, ви можете потрапити за умови що у вас є 5к гс (в залежності від рейду), зі всіма інчантами та гемами.

Якщо вас беруть на основне РТ в ЦЛК / РС / ТОГК, але ваш ГС менше 5.2к, або немає повністю інчантів, гемів, то ви потрапляєте за умови що ви йдете лише за ДКП, та баджами, тобто бідати речі вам дозволено лише у випадку якщо вони нікому не потрібні.

Якщо у вас лише один персонаж, або ви фізично не можете прийти у інші дні на РТ, попередьте про це коли будете записуватись на РТ (див. розділ [F.A.Q.](./f-a-q.md)) для того щоб РЛ міг вирішити питання по компонуванню рейду.

## Перелік необхідних аддонів для комфортного проходження РТ:

1. Pally Power - аддон виключно для паладінів та РЛів, який дозволяє виставити бафи, та аури паладінам в рейді
2. Deadly Boss Mode (DBM) - аддон для гравців в рейді який анонсує механіки босів
3. Будь який аддон для рейд фреймів Grid2, VuhDo, XPearl тощо. - обов’язково щоб було відображення фреймів рейду, для ситуаційної обізнаності
