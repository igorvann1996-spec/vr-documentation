# Загальні положення

## Ідея гільдії

Гільдія "Verkhovna Rada" створена для розвитку українського ком'юніті на сервері Warmane Icecrown. Ми об'єднуємо гравців, які прагнуть до якісного PvE-контенту, взаємодопомоги та приємного спілкування.

## Цілі гільдії

1. Створити дружнє та самодостатнє українське комюніті для приємної комунікації та спільного проведення часу
2. Спільний PvE прогрес, та одягання персонажів
3. Проведення PvE та PvP івентів з іншими українськими гільдіями для підтримки українського ком'юніті на сервері
