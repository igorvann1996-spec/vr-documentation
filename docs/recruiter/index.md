# 📜 Загальні Положення Міністерства Кадрів

Рекрутери є офіційними представниками Держави та обличчям Гільдії «Верховна Рада». 
Їхня місія — залучення нових громадян до лав Збройних Сил, забезпечення стабільного притоку бойових одиниць та підтримка іміджу гільдії на міжнародній арені.
Відповідальність за первинний інструктаж та видачу посібників новобранцям лежить на співробітниках цього Міністерства.
