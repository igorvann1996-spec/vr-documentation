# 📜 Протокол Отримання Темної Скорботи (Shadowmourne)

Цей офіційний протокол регламентує порядок видачі та збору уламків для легендарної зброї — Темної Скорботи ([Shadowmourne](https://www.wowhead.com/wotlk/item=49623/shadowmourne)). Зважаючи на колосальну бойову міць цього артефакту, процес жорстко контролюється Командуванням.

## 📝 Постановка на Облік (Черга)

Для того щоб отримати державне фінансування та дозвіл на збір уламків (шардів) або виконання квестів, боєць повинен відповідати наступним критеріям:
1. Мати державне звання не нижче ніж **Рейдер**.
2. Подати запит Голові Верховної Ради (ГМ) або уповноваженим офіцерам, вказавши ім'я персонажа та поточну кількість зібраних уламків/стадію квесту.

**Формат рапорту:** `{ім’я персонажа} {к-ть шардів}` 
*(Наприклад: "Logansdk 15")*

## ⚔️ Правила Мобілізації для Збирачів

Під час реєстрації на бойову операцію (РТ), боєць, який знаходиться в офіційному списку на збір, **зобов'язаний** вказати свій статус.
*Приклад запису:*
```
Loganshunt mm 6.5
Logansdk frost 6.3 - шарди
```

## ⚖️ Дисципліна та Санкції

Збір артефакту — це привілей, який вимагає постійної явки на військові операції.

1. **Дезертирство:** Якщо боєць пропускає дві або більше військових кампаній (РТ) поспіль без попередження, він **автоматично виключається** з офіційної черги. Щоб відновити збір, йому доведеться ставати в чергу заново.
2. **Оновлення Даних:** Після кожного успішного рейду ГМ зобов'язаний зафіксувати прогрес (оновити кількість уламків/квестів в інформаційному каналі).
3. **Ротація:** Після того як боєць завершує створення Темної Скорботи, він знімається з обліку, а інші кандидати автоматично просуваються вище по черзі.

## 🛑 Державні Обмеження

> * **Монополія:** Одночасно один громадянин може збирати уламки **ЛИШЕ ДЛЯ ОДНОГО** персонажа.
> * **Ліміт Реєстру:** В офіційному державному реєстрі на збір Темної Скорботи може перебувати **МАКСИМУМ 5 (п'ять)** бійців одночасно.
